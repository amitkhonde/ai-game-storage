# Game Design Document ## Expanded Theme: "Cipher Nexus: Mind Reader" A mysterious text-based riddle game where players solve increasingly complex word puzzles, cryptic clues, and logic challenges. The game analyzes their solving patterns, reaction times, and approach strategies to reveal deep personality insights. Each riddle is presented with animated glyphs, morphing text, and particle effects that respond to player interactions. ## Core Gameplay Loop 1. **Goal**: Solve text-based riddles by typing answers or selecting from animated word fragments 2. **Challenge**: Riddles increase in complexity - from simple wordplay to cryptic logic puzzles 3. **Feedback**: Immediate visual feedback with particle explosions, text morphing, and color shifts 4. **Reward**: Unlock new riddle types and see personality analysis build in real-time ## Game Systems ### Physics & Particles - **Floating text fragments** that players can drag and combine - **Particle explosions** on correct answers (success particles) - **Glitch effects** on wrong answers with screen shake - **Morphing letter animations** using physics-based transitions ### UI & Visual Design - **Breathing UI elements** with subtle scale animations - **Dynamic color palette** that shifts from cool blues to warm golds as difficulty increases - **Layered background** with animated geometric patterns - **Floating glyph particles** constantly moving in the background ### Input Handling - **Text input** for direct answer typing - **Drag & drop** for word fragment puzzles - **Tap/click** for multiple choice elements - **Gesture recognition** for drawing symbols on mobile ### Analytics & Behavioral Tracking - **Reaction Time**: Speed of first interaction with each riddle - **Solving Strategy**: Linear vs exploratory approach - **Persistence**: Retry frequency and time spent per riddle - **Risk Taking**: Tendency to guess vs careful consideration - **Pattern Recognition**: Speed of identifying riddle types ## Progression System - **Phase 1 (0-30s)**: Simple word association riddles - **Phase 2 (30-60s)**: Pattern recognition and logic puzzles - **Phase 3 (60-90s)**: Complex cryptic riddles requiring lateral thinking - **Visual Evolution**: Background complexity and particle density increase with each phase ## Instruction Screen Animated SVG panel featuring: - **Morphing text examples** showing how riddles transform - **Interactive word fragments** players can immediately try - **Glowing pathway** showing the progression from simple to complex - **Personality meter preview** hinting at the final reveal ## Art & Animation Style - **Cyberpunk-inspired** color scheme with neon accents - **Constant text morphing** and letter-by-letter reveals - **Particle trails** following cursor/finger movements - **Breathing UI elements** with 2-3 second pulse cycles - **Dynamic lighting effects** that respond to player actions ## Behavioral Mapping - **Quick + Accurate** = "Lightning Analyst" - **Slow + Methodical** = "Deep Thinker" - **High Retry Rate** = "Persistent Solver" - **Pattern Jumper** = "Intuitive Genius" - **Risk Taker** = "Bold Decoder" ## End Reveal Screen Animated personality profile with: - **Morphing title** revealing their solver archetype - **Animated stats** showing their unique
